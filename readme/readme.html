Инструкция

Первичная настройка программы

1. API key

Сначала необходимо проверить наличие API ключа, который находится в директории с программой в файле api_key.txt

Если ключа нет, его необходимо взять из аккаунта SE Ranking. Подробнее тут в блоке "Получение API-ключа".

2. Установка python и необходимых модулей

Необходимо зайти на официальный сайт Python и скачать версию 3.7+. Чтобы проверить корректность установки, необходимо зайти в терминал и ввести команду python --version

В программе используется модуль pandas. Его необходимо установить отдельно.

Для установки модуля pandas необходимо зайти в терминал и ввести команду pip install pandas, если не срабоатет можно попробовать ввести команду pip3 install pandas.
